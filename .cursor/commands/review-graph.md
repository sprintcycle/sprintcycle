# LangGraph review / LangGraph 审查

Use the `graph-orchestrator` subagent to review graph nodes, state transitions, dispatch flow, and plan/sprint split behavior.

使用 `graph-orchestrator` 子代理审查 graph 节点、状态流转、调度流程与 plan/sprint 拆分行为。

## When to use / 适用场景
- LangGraph runtime changes / LangGraph runtime 变更
- Node responsibility changes / 节点职责变更
- Stage transition changes / 阶段流转变更
- Dispatch or routing changes / 派发或路由变更
- Graph orchestration boundary concerns / 编排边界风险

## Expected output / 期望输出
- Summary
- Flow analysis
- Blocking issues
- Warnings
- Suggestions
- Verdict
