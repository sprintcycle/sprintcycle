# Lifecycle review / 生命周期审查

Use the `lifecycle-auditor` subagent to review execution lifecycle, runtime linkage, observability, and evolution/promotion consistency.

使用 `lifecycle-auditor` 子代理审查执行生命周期、运行时联动、可观测性以及演化/晋升一致性。

## When to use / 适用场景
- Execution lifecycle changes / 执行生命周期变更
- Runtime registry changes / runtime registry 变更
- Observability changes / 可观测性变更
- Promotion or evolution policy changes / 晋升或演化策略变更
- Trace / state consistency concerns / trace 与 state 一致性风险

## Expected output / 期望输出
- Summary
- Lifecycle risks
- State consistency issues
- Suggestions
- Verdict
