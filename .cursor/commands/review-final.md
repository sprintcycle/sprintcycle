# Final review summary / 最终审查汇总

Use the `review-commander` subagent to consolidate all specialist review findings into a final decision.

使用 `review-commander` 子代理汇总各专岗审查结论并给出最终决定。

See `docs/CURSOR_TEAM_PLAYBOOK.md` for the review workflow and role boundaries.

## When to use / 适用场景
- Large changes spanning multiple layers / 跨多层的大改动
- PRs requiring a single final verdict / 需要统一最终结论的 PR
- Changes reviewed by multiple specialist subagents / 已由多个专岗审查的变更

## Expected output / 期望输出
- Executive summary
- Consolidated findings
- Action items
- Verdict
